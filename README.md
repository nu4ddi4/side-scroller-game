# 횡스크롤 게임

코딩동아리 모둠별 발표회와 학교 축제 시연용으로 만든 간단한 횡스크롤 점프 게임 데모입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 조작법

- 이동: 방향키 왼쪽/오른쪽 또는 A/D
- 점프: 스페이스, W, 위쪽 방향키
- 목표: 장애물을 피해서 초록색 골인지점까지 이동

## 구현 기능

- 좌우 이동
- 점프
- 중력
- 발판 충돌 판정
- 장애물 충돌 판정
- 아이템 점수
- 카메라 스크롤
- 골인 판정
- 게임 오버 / 클리어 화면

## 발표 설명용 핵심

이 게임은 HTML의 canvas를 이용해 화면을 그리고, JavaScript로 플레이어의 위치와 속도, 충돌 판정을 처리합니다.
플레이어가 발판 위에 있을 때만 점프할 수 있게 만들었고, 장애물과 닿으면 게임 오버, 골인지점에 도착하면 클리어가 됩니다.
